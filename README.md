# Agent Skills

Agent Skills are folders of instructions, scripts, and resources that AI agents can discover and use to perform at specific tasks. Write once, use everywhere.

## Installing 

### Claude Code 

Run `/plugin` in the CLI, and select `Add Marketplace`. Input `moonbitlang/skills` to add this marketplace. 

### Codex 

Run in the CLI: `$skill-installer install <skill-name> by .claude-plugin/marketplace.json from https://github.com/moonbitlang/skills.git`
